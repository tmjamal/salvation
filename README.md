## 📖 About
A complete Islamic Malayalam content management system featuring:
- 📚 **Articles** - Islamic teachings in Malayalam
- 📖 **Stories** - Islamic stories and teachings
- 🕌 **Admin Panel** - Easy content management
- 🎨 **Modern Design** - Responsive and beautiful

## 🚀 Technologies Used
- **Flask** - Python web framework
- **SQLAlchemy** - Database ORM
- **Render** - Free hosting platform
- **SQLite** - Database (free forever)

## 🎯 Features
- ✅ **Admin Panel** - Add/edit content
- ✅ **Image Support** - Upload images for stories/articles
- ✅ **Category Management** - Organize content
- ✅ **Responsive Design** - Works on all devices
- ✅ **Malayalam Content** - Full Malayalam support
